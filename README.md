# Colección de ejercicios de introducción a Python

Este repositorio contiene una serie de ejercicios prácticos para consolidar los conceptos básicos de programación en Python.

**Nota Importante:** los ejercicios estan en la rama **`master`**

## Contenido

- **ejercicio_01.py**: Manejo de edades para licencias de conducción.
- **ejercicio_02.py**: Clasificador de calificaciones escolares.
- **ejercicio_03.py**: Sistema básico de manejo de usuarios.
- **ejercicio_04.py**: Sistema de descuentos en una tienda.
- **ejercicio_05.py**: Sistema básico de cajero automático.
- **ejercicio_06.py**: Simulador de semáforo.
- **ejercicio_07.py**: Sistema básico de verificación de contraseña.
- **ejercicio_08.py**: Clasificación de temperatura.
- **ejercicio_09.py**: Sistema de menú de restaurante.
- **ejercicio_10.py**: Conversor de monedas.
